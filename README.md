# Ultrasonic Sensor Reading with ROS 2

This repository contains the source code for reading data from an ultrasonic distance sensor, serving as the foundational perception module for an Autonomous Mobile Robot (AMR) project.

---

## Hardware Architecture
* **Microcontroller:** Arduino / ESP32 (or similar compatible board)
* **Sensor:** HC-SR04 Ultrasonic Distance Sensor

---

## Software Framework & Standards
* **Operating System:** Linux Ubuntu
* **Middleware:** ROS 2 (Robot Operating System)
* **Code Styling:** Google Style Guide / Clean Code principles
* **Version Control:** Git & GitHub Workflow

---

## Documentation & Methodology
This project strictly follows global engineering practices by implementing discrete modular pipelines. Every sub-module undergoes complete functional testing via simulation before deployment to physical hardware.
